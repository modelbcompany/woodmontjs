# Fieldset

`<fieldset>` element with the class `adm-fieldset`.  

Because `<fieldset>` elements ignore Flexbox properties, a `Container` component
with the class `fieldset-container` will wrap the child elements.

**https://developer.mozilla.org/docs/Web/HTML/Element/fieldset**  
**https://khrome.dev/blog/html-elements-with-flex-box-quirks/**
