# Select

`<div>` element with the class `adm-select`.

Inside of the container, a native `<select>` element will be rendered, as well
a custom dropdown `Icon`.

Props passed to this component will be passed to the native `<select>` element,
**NOT** the outer `<div>` element.

**https://developer.mozilla.org/docs/Web/HTML/Element/select**
