##### Accessibility

An `Icon` has two use cases:

- **Decorative**: These icons are only used for visual or branding
  reinforcement. If they were removed from the page, users would still understand and be able to use your page.

- **Semantic**: These icons are only used to convey meaning. This includes icons
  without text next to them used as interactive controls â€” buttons, form
  elements, toggles, etc. Icons without text next to them should use the
  `aria-label` attribute.

Reference: https://material-ui.com/components/icons/#accessibility
