# Typography

Families: IBM Plex Mono, Swiss 721

<br/>

## Usage

<br/>

### HTML

```html
<!-- IBM Plex Mono -->
<link type="text/css" rel="stylesheet"
  href="//fonts.googleapis.com/css2?family=IBM+Plex+Mono:wght@200;300;400;500;600;700&display=swap" />

<!-- Swiss 721 -->
<link type="text/css" rel="stylesheet" href="//fast.fonts.net/cssapi/ea5e2124-bbdb-4e91-8a98-9b549b432fdb.css" />
```

### Sass

```scss
/// Primary font family
/// @type {List}
$primary-font-family: IBM Plex Mono, monospace

/// Secondary font family
/// @type {List}
$secondary-font-family: Swiss_721 W01, sans-serif
```
